1. Les Nombres Complexes

1.1. Définition

Un nombre complexe est un nombre qui est la somme d'un nombre réelle et d'un nombre réelle multiplié par \(\color{#00bf00}i\) (tel que \(\color{#00bf00}i^{\color{#ffffff}2}\color{#ffffff}=-1\)), donc si \(z\) est un nombre complexe, alors: \[z=a+b\times\color{#00bf00}i\color{#ffffff}\] Les nombres complexes sont utilisés en physique, électricité, et la rotation.

1.1.1. Opérations

Soient \(z_1 = a + b \color{#00bf00}i\color{#ffffff}\) et \(z_2 = c + d \color{#00bf00}i\color{#ffffff}\), alors :

  • Addition : \(z_1 + z_2 = (a + c) + (b + d) \color{#00bf00}i\color{#ffffff}\)
  • Soustraction : \(z_1 - z_2 = (a - c) + (b - d) \color{#00bf00}i\color{#ffffff}\)
  • Multiplication : \(z_1 \times z_2 = (ac - bd) + (ad + bc) \color{#00bf00}i\color{#ffffff}\)
  • Division : \[\n \frac{z_1}{z_2} = \frac{a + b \color{#00bf00}i\color{#ffffff}}{c + d \color{#00bf00}i\color{#ffffff}}\n = \frac{(ac + bd) + (bc - ad) \color{#00bf00}i\color{#ffffff}}{c^2 + d^2}\n \]

1.1.2. Forme rectugulaire et polaire

Un nombre complexe peut être représenté par sa forme rectangulaire: \[ z=a+b\times\color{#00bf00}i\color{#ffffff}=a+b\color{#00bf00}i\color{#ffffff} \] mais il peut être aussi représenté sous sa forme polaire \[ z = r e^{\color{#00bf00}i\color{#ffffff}\theta} = \begin{cases} \sqrt{a^{2}+b^{2}} e^{\color{#00bf00}i\color{#ffffff}\arctan{\frac{b}{a}}} & a > 0 \\ \sqrt{a^{2}+b^{2}} e^{\color{#00bf00}i\color{#ffffff}\left(\arctan{\frac{b}{a}} + \pi\right)} & a < 0, b > 0 \\ \sqrt{a^{2}+b^{2}} e^{\color{#00bf00}i\color{#ffffff}\left(\arctan{\frac{b}{a}} - \pi\right)} & a < 0, b < 0 \\ \sqrt{b^{2}} e^{\color{#00bf00}i\color{#ffffff}\frac{\pi}{2}} & a = 0, b > 0 \\ \sqrt{b^{2}} e^{-\color{#00bf00}i\color{#ffffff}\frac{\pi}{2}} & a = 0, b < 0 \end{cases} \] Ici, \(\arctan\) est en radian, donc un tour complet est \(2\pi\) et non \(360\). \(\arctan{\frac{b}{a}}\) est l'argument de \(z\), alors que \(\sqrt{a^2+b^2}\) est sa valeur abosule.

1.1.3. Utiliser les nombres complexes pour la rotation

La multiplication par un nombre complexe de la forme \(e^{\color{#00bf00}i\color{#ffffff}\theta}\) correspond à une rotation du plan complexe d'un angle \(\theta\) autour de l'origine. Si \(z\) représente un point du plan, alors le point \(z'\) obtenu après rotation de \(\theta\) radians est : \[ z' = e^{\color{#00bf00}i\color{#ffffff}\theta} \; z. \] En particulier, la multiplication par \(\color{#00bf00}i\color{#ffffff}=e^{\color{#00bf00}i\color{#ffffff}\frac{\pi}{2}}\) effectue une rotation de \(+90^\circ\) (\(+\frac{\pi}{2}\) radians) dans le sens anti-horaire, et par \(e^{-\color{#00bf00}i\color{#ffffff}\frac{\pi}{2}}\) une rotation de \(-90^\circ\).

Exemple : si \(z=1+0\color{#00bf00}i\color{#ffffff}\) (le point \((1,0)\)), alors \[ z' = e^{\color{#00bf00}i\color{#ffffff}\frac{\pi}{2}}(1) = \color{#00bf00}i\color{#ffffff} \] qui correspond au point \((0,1)\).

Pour faire une rotation autour d'un point \(z_0\) qui n'est pas l'origine, on translate d'abord pour ramener \(z_0\) à l'origine, on effectue la rotation, puis on translate de retour : \[ z' = (z - z_0) e^{\color{#00bf00}i\color{#ffffff}\theta} + z_0. \]

Cette représentation est très pratique en infographie et en géométrie: elle combine la rotation et la dilation (multiplication par \(r e^{\color{#00bf00}i\color{#ffffff}\theta}\) multiplie la distance au centre par \(r\) et effectue une rotation de \(\theta\)).
